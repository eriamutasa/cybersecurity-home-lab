FINDING 3 — FTP (Port 21)
Filter: ftp
Result: HIGH — Anonymous login accepted with blank password.
        USER ftp / PASS (blank) visible in TCP stream.
        No credentials required to access file system.
        Mirrors the exact class of vulnerability in the
        2021 Twitch breach (unauthenticated file access).
Risk: HIGH
Recommendation: Disable anonymous FTP. Replace with SFTP.

KEY LESSON:
Protocol choice is a security decision not just a
technical preference. Telnet, FTP, and HTTP expose
everything to anyone watching the network.
SSH, SFTP, and HTTPS encrypt all data in transit.
A SOC analyst must understand this distinction to
detect credential interception attacks in real time
and provide clear remediation guidance.
